# Manifestation Travel Diary

A GitHub Pages site for Luciana and her husband’s 2029–2034 travel manifestation timeline.

## Structure

- `site/index.html` — static landing page
- `site/styles.css` — visual design
- `site/app.js` — renders timeline from JSON
- `site/data/journey.json` — destination analysis, completed anchors, and faux diary timeline

## Local preview

```bash
cd site
python3 -m http.server 8765
```

Then open <http://localhost:8765>.
